My ZSH Profiles
===============

For installation clone this repo as a new subdir in $HOME and link `.zshenv`
into your home directory.

    $ cd
    $ git clone ......
    $ ln -sf ~/.zsh/.zshenv
    $ exec $SHELL -l

Customizations
--------------

    $ cd ~/.zsh
    $ cp .zshrc.local.sample .zshrc.local
    $ cp .zshrc.path.sample .zshrc.path
    $ vi .zshrc.local .zshrc.path
