My ZSH Profiles
===============

For installation clone this repo as a new subdir in $HOME and link `.zshenv`
into your home directory.

    $ cd
    $ git clone ...... .zsh
    $ ln -sf ~/.zsh/zshenv .zshenv
    $ exec $SHELL -l

Customizations
--------------

    $ cd ~/.zsh
    $ cp local.zsh.sample local.zsh
    $ cp path.zsh.sample  path.zsh
    $ vi local.zsh path.zsh
